## Quick Reference

### Brand Colors

| Role | Hex | Usage |
|------|-----|-------|
| **Primary** | `#0066FF` | Electric blue — CTAs, highlights |
| **Secondary** | `#6366F1` | Indigo — supporting elements |
| **Accent** | `#F472B6` | Pink — emphasis, delight |
| **Navy** | `#0A1628` | Background base |
| **Success** | `#10B981` | Revenue recovered, positive |
| **Warning** | `#F59E0B` | Costs, alerts |

### Typography

- **Primary**: Geist Sans (variable)
- **Mono**: Geist Mono (code, data, numbers)
- **Display**: 72px / weight 700 / tracking -2%
- **H1**: 48px / weight 600 / tracking -1.5%
- **H2**: 36px / weight 600 / tracking -1%
- **Body**: 16px / weight 400 / line-height 1.6

### Brand Archetypes

- **Primary**: The Sage (65%) — Knowledge, expertise, truth-seeking
- **Secondary**: The Ruler (35%) — Control, order, infrastructure

### Tagline

"Billing infrastructure for AI SaaS"
