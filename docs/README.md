## Naming Convention

All documentation files follow **kebab-case** naming:
- ✅ `brand-guidelines-ai-rio-2025-02-10.md`
- ✅ `phase-5-summary.md`
- ✅ `blog-setup-guide.md`
- ❌ `Brand_Guidelines.md`
- ❌ `PHASE_5_SUMMARY.md`

## Contributing to Documentation

When adding new documentation:

1. Determine the appropriate folder (brand, development, guides, research, or references)
2. Use kebab-case naming
3. Update the relevant README.md index
4. Cross-link related documents
